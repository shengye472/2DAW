# ANGULAR CHULETA

## Comandos ng
``` bash
ng new <nombre>     # Crear nuevo proyecto de angular
ng serve            # Iniciar servidor de desarrollo
ng build            # Compilar proyecto

ng g c <nombre>     #ng generate component Components/<nombre>
ng g i <nombre>     #ng generate interface Model/<nombre>
ng g s <nombre>     #ng generate service Services/<nombre>
ng g g <nombre>     #ng generate guard Guards/<nombre>
```
